# Contributing

Спасибо за интерес к проекту.

## Требования

- Node.js (рекомендуется LTS)
- npm

## Локальный запуск

```bash
npm install
npm run dev
```

## Проверки перед PR

- Линтинг:

```bash
npm run lint
```

- Сборка:

```bash
npm run build
```

## Стиль кода

- Старайся придерживаться существующего стиля проекта.
- Не добавляй большие рефакторинги в PR без отдельного обсуждения.

## Как предлагать изменения

- Для багов: приложи шаги воспроизведения, ожидаемое/фактическое поведение, скриншоты (если применимо).
- Для фич: опиши мотивацию и UX.
